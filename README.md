# Chronos

Aplicacao Kanban para atividades pessoais, agora evoluida para React com Vite.

## Funcionalidades

- Criar, editar, excluir e mover tarefas entre colunas.
- Filtros por texto e prioridade.
- Indicadores de tarefas, alta prioridade, vencimentos proximos e concluidas.
- Tema claro/escuro com persistencia local.
- Importacao e exportacao do quadro em JSON.
- Salvamento automatico no `localStorage`.

## Como rodar

```bash
npm install
npm run dev
```

Acesse `http://127.0.0.1:5173/`.

## Build de producao

```bash
npm run build
```
